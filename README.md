# Hash Table in C

> [!WARNING]
> This implementation is deprecated. Please use [sptl.h](https://github.com/onlyspxctre/sptl.h), which contains the most up-to-date implementation of this hash table.

Simple hash table implementation in pure C. Uses the [FNV Hash algorithm by Glenn Fowler and Phong Vo.](http://isthe.com/chongo/tech/comp/fnv/#FNV-1a)
